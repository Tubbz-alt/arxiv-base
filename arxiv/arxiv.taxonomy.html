arxiv.taxonomy package¶

arXiv group, archive and category definitions.

arxiv.taxonomy.CATEGORY_ALIASES = {'math.IT': 'cs.IT', 'math.MP': 'math-ph', 'q-fin.EC': 'econ.GN', 'stat.TH': 'math.ST'}¶

canonical category

This model is based on the notion that only two categories may be equivalent–not more. There would have to be some significant changes to the (classic) code to support three-way equivalences.
